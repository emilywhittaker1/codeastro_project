Differential Photometry Functions¶

Function to perform correlations.

Input: fits image, stellar coordinates Output: lightcurve

differential_photometry.star_counts(pixels, data)¶

Star Counts

Using aperture photometry to find the number of counts measured from a star.

Parameters

  • pixels (array) – numpy vector. Location of the target star in pixels.

  • data (array) – numpy 2D array. Image data of the star from the fits file.

Returns

instrument counts from the star

Return type

float
